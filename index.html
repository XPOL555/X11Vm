Display Configuration

X11Vm uses Xpra to seamlessly forward native Linux windows (like XEmacs or Xterm) from the Docker container directly to your host operating system. To enable this feature, you must have Xpra installed on your Mac or PC.

  • On Mac, you can install it via Homebrew: brew install xpra or download it from xpra.org.
  • On Windows, download the installer from xpra.org.

Client DPI

Force Xpra to use a specific DPI (e.g. 96, 120, 144) to fix scaling warnings. Leave at 0 for auto-detection.

Welcome to X11Vm

X11Vm allows you to run a Linux desktop environment on your system via Docker, seamlessly forwarding windows to your native screen using Xpra.

1. Build the Image

Click 'Build Image' to compile the Docker container.

2. Install Xpra

Make sure Xpra is installed on your host system to display the windows.
